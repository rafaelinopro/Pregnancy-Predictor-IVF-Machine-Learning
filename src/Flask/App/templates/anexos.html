(*) Este error es indeseable

El modelo seleccionado se entrenó previa reducción de feautures con RFE (Recursive Feauture Election), utilizando como estimado DecisionTreeClassifier.
Luego se aplicó un balanceado parcial de un 33% para la clase minoritaria, y de un 66% para la clase mayoritaria.
De esta manera se consiguieron las predicciones mas equilibradas. Si se forzaba alguna métrica en particular, la matriz de confusión se despalazaba y caian otras.
Entonces, analizamos las opciones tratando de que los falsos positivos se mantengan en un porcentaje aceptable.
